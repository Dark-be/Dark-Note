#### SSH
* 使用SSH密钥连接更稳定，详见[Connecting to GitHub with SSH](https://docs.github.com/en/authentication/connecting-to-github-with-ssh)
* 更改私钥密码
>ssh-keygen -f id_rsa -p

#### Work together
* 一般遵循以下流程
> git add .
git commit -m "new commit"
